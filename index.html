⚙️ Supabase 配置

在 supabase.com 创建项目后，将以下信息填入并保存。配置保存在浏览器 localStorage 中。

📖 使用指南

1. 注册 Supabase 免费账号，创建新项目

2. 在 Supabase SQL Editor 执行 supabase-schema.sql 建表

3. 在 Supabase Storage 创建名为 media 的 Public Bucket

4. 在 Supabase Dashboard → Settings → API 获取 URL 和 Anon Key

5. 在此页面填写配置并保存

6. 将 dist 目录推送到 GitHub 仓库，开启 GitHub Pages

7. 部署后将 GitHub Pages 地址填入配置中的「Site URL」
